## Flapp Bird Clone

![Alt text](https://images.br.sftcdn.net/br/scrn/69675000/69675957/image-06-700x393.jpg "Flappy Bird")


## Motivação

Projeto criado como exercício do [curso de Android] (https://www.udemy.com/curso-completo-do-desenvolvedor-android/learn/v4/overview).
Trata-se do famoso jogo Flappy Bird.

## Linguagem

A linguagem utilizada é Java para Android, com utilização da biblioteca [libgdx] (http://libgdx.badlogicgames.com/) para construção dos gráficos e figuras.
